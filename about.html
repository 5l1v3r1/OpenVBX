About Us

OpenVBX is an open source project started and powered by Twilio cloud APIs. OpenVBX was launched to give developers an easy way to to build powerful phone systems that look great. OpenVBX extends Twilio’s API by providing a user interface and other primitives like voicemail that build upon Twilio’s core API stack. OpenVBX is designed to be extensible by developers, and ease to use by end users. Currently OpenVBX is a 100% open source project.
