Installing OpenVBX

  1. Get the Code

    Download the latest release and unpack it

  2. Create a new MySQL database

    and a MySQL database user, too

  3. Run the installer

    Navigate your browser to where you unpacked OpenVBX

  4. Connect to Twilio

    Register for a Free Twilio Account

  5. Login

    to OpenVBX and enjoy!

Detailed Instructions can be found in our Install Guide

Upgrading OpenVBX

  1. Get the Code

    Upload the latest release to your server

  2. Upgrade Database

    Navigate to [OpenVBX URL]/upgrade

Detailed install instructions can be found in our Upgrade Guide
